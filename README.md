# Kotoba
スマホ × 日本語フリック プログラミングに適した自分が使いやすい言語、およびコンパイラプロジェクト

**Kotoba** は、日本語フリック入力に最適化された  
**独自プログラミング言語とそのコンパイラ／仮想マシン**です。

スマートフォンだけでの入力・実装・実行を前提に設計されており、  
記号依存・英単語依存・過剰な構文規則を意図的に排除しています。

> 書けることより、**打てること**を優先する  
> 正しさより、**意味に近いこと**を優先する

---

## 特徴

- 📱 **スマホ最適化**
  - ひらがなキーワード中心
  - 記号を最小限に削減
  - インデント不要（`:` と `おわり` によるブロック構造）

- 🇯🇵 **日本語ファースト**
  - 日本語キーワード
  - 日本語識別子（UTF-8）
  - 思考と言語入力の距離を縮める設計

- ⚙️ **本気の処理系**
  - 字句解析 → 構文解析 → バイトコード生成
  - 独自VMによる実行
  - C++ によるフルスクラッチ実装

- 🧠 **口語志向**
  - 会話文に近い構文
  - 条件・繰り返し・関数を自然言語的に表現
  - 技術デモで終わらず、作品・研究に転用可能
